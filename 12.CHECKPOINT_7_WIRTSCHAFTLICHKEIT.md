# CHECKPOINT 7 — WIRTSCHAFTLICHKEIT (BehördenReply)

## Ziel

Nachweis, dass BehördenReply mit einem realistischen Preismodell wirtschaftlich betrieben werden kann.

---

# 1. Kosten pro Anfrage

## Verwendetes Modell

OpenAI GPT-4.1-mini

Begründung:

- ausreichend leistungsfähig für Behördenkommunikation
- deutlich günstiger als größere Modelle
- für MVP wirtschaftlich sinnvoll

---

## Geschätzter Tokenverbrauch pro Dokument

BehördenReply verarbeitet:

1. hochgeladenes Schreiben
2. interne Analyse
3. Rückfragen
4. Antworten des Nutzers
5. finale Antwort

Konservative Schätzung:

### Input

- Dokument: 2.000 Token
- Rückfragen + Antworten: 1.500 Token

Gesamt Input:

3.500 Token

### Output

- Analyse
- Rückfragen
- finale Antwort

Gesamt Output:

1.500 Token

---

## Kosten pro Dokument

Annahme:

- 3.500 Input-Token
- 1.500 Output-Token

Geschätzte KI-Kosten:

ca. 0,005 € pro vollständig bearbeitetem Dokument

(≈ 0,5 Cent)

Für Investorengespräche wird bewusst mit einem Sicherheitsaufschlag gerechnet.

Deshalb:

Rechenwert pro Dokument:

0,01 €

(1 Cent)

---

# 2. Nutzungsannahmen

## Free-Nutzer

1 kostenloses Dokument

Durchschnitt:

1 Dokument pro Monat

KI-Kosten:

0,01 € pro Nutzer und Monat

## Pay-per-Use-Nutzer

Durchschnitt:

3 Dokumente pro Jahr

≈ 0,25 Dokumente pro Monat

## Abo-Nutzer

bis zu 30 Dokumente pro Monat

Konservative Annahme:

durchschnittlich 10 Dokumente pro Monat

KI-Kosten:

0,10 € pro Monat

---

# 3. Conversion-Annahme

5 % der registrierten Nutzer werden zahlende Kunden.

Aufteilung:

- 80 % Pay-per-Use
- 20 % Abo

# 4. Fixkosten

- Konservative Gesamtfixkosten: 20 € / Monat
