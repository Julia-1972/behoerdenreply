# Abschlussprojekt – распределение работы между ИИ

## Цель

Максимальная оценка за Abschlussprojekt при минимальном риске и эффективном использовании разных ИИ-инструментов.

---

## Рекомендуемое распределение

| Checkpoint | Основной исполнитель |
|------------|---------------------|
| CP1 – Idee & Projekt-Steckbrief | ChatGPT |
| CP2 – Zielgruppe & Business Case | ChatGPT |
| CP3 – Das Produkt | Claude Code |
| CP4 – Landing Page (структура и тексты) | ChatGPT |
| CP4 – Landing Page (реализация) | Claude Code |
| CP5 – Marketing / Ads | ChatGPT |
| CP6 – Recht & Compliance | ChatGPT |
| CP7 – Wirtschaftlichkeit | ChatGPT |
| CP8 – Pitch | ChatGPT |
| CP9 – Deploy | Claude Code |

---

## ChatGPT

Использовать как:

- продуктового консультанта;
- инвестора;
- экзаменатора;
- бизнес-аналитика.

Лучше всего подходит для:

### CP1
- выбор идеи;
- формулировка проблемы;
- позиционирование;
- Steckbrief.

### CP2
- Persona;
- Marktgröße;
- Pricing;
- Freemium;
- Business Case.

### CP4
- Hero;
- Nutzenversprechen;
- Struktur Landingpage;
- Pricing-Blöcke;
- CTA.

### CP5
- выбор рекламного канала;
- Pain Hooks;
- Ad Angles;
- A/B-Test-Logik;
- тексты объявлений.

### CP6
- Impressum;
- Datenschutzerklärung;
- AGB;
- GDPR;
- EU AI Act;
- Compliance Check.

### CP7
- Token-Kalkulation;
- Kosten pro Anfrage;
- Szenarien 100 / 1.000 / 10.000 Nutzer;
- Profitabilität;
- Investor Questions.

### CP8
- Elevator Pitch;
- Storyline;
- Jury-Fragen;
- Präsentationsstruktur.

---

## Claude Code

Использовать как:

- ведущего разработчика проекта.

Лучше всего подходит для:

### CP3
- Next.js;
- TypeScript;
- Supabase;
- Auth;
- Dashboard;
- Free/Paid Logic;
- Prompt Integration;
- Fehlerbehebung.

### CP4
- реализация Landing Page.

### CP9
- Deployment;
- Vercel;
- Environment Variables;
- финальное тестирование.

---

## Claude Project

Использовать как:

- центральное хранилище знаний проекта.

Лучше всего подходит для:

- хранения контекста;
- анализа старых материалов;
- поиска противоречий;
- сверки решений;
- накопления документации.

Не рекомендуется использовать как единственный инструмент для принятия бизнес-решений по Abschlussprojekt.

---

## Итоговая стратегия

ChatGPT:
- CP1
- CP2
- CP4 (контент)
- CP5
- CP6
- CP7
- CP8

Claude Code:
- CP3
- CP4 (реализация)
- CP9

Claude Project:
- база знаний;
- архив проекта;
- контроль согласованности решений.

Идея: содержательная часть проекта разрабатывается через ChatGPT, программирование выполняется через Claude Code, долговременный контекст хранится в Claude Project.
