# System Overview --- BehördenReply

## Core Idea

User uploads a German official letter → AI asks clarifying questions →
AI generates final German response.

## Input

-   PDF only

## Process

1.  Upload document
2.  AI analyzes content
3.  AI generates minimal necessary questions
4.  User answers step-by-step
5.  AI re-evaluates after each answer
6.  When enough data → final response

## Output

-   German official-style letter
-   downloadable PDF + DOCX
-   shown on screen

## AI Behavior

-   Always minimal questions
-   Always context-aware
-   Always decision-based (not fixed flow)
-   Stop when confident enough

## Failure Handling

-   If unreadable → error + retry upload
